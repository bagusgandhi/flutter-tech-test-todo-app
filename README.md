# Todo App

## Getting Started

## Installation
To setup this project, follow these steps:
```bash
# Flutter version 3.19.2
# Dart version 3.3.0

# clone the repository
$ git clone https://github.com/bagusgandhi/flutter-tech-test-todo-app.git

# navigate to repository
$ cd ./flutter-tech-test-todo-app

# install dependency
$ flutter pub get

```

## Set config Mock API URL
inside <b>config.dart</b> you need to fille this url mock api

```dart
class AppConfig {
  static const String apiUrl = 'https://65e97981c9bf92ae3d393ceb.mockapi.io/api/v1';
}

```

for mock api using mockapi.io


## Running the App

Use the following command to run the app on an emulator or connected device:

```bash

$ flutter run

```
